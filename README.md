# Encontre o Menor Caminho na Ibiapaba
## Descrição
Este aplicativo tem o objetivo de descobrir de forma prática o menor caminho entre as cidades da microrregião da Serra da Ibiapaba no Ceará.
## Tecnologias
* HTML
* CSS
* JavaScript
